                          InterProScan Domain Model

  What is it?
  -----------

  InterProScan Domain Model represents the domain model for the InterProScan project.

  Requirements
  ------------

  To install:
   o  A Java 1.5 or later compatible virtual machine for your operating system.
   o  Maven 2.0 or later

  To run:
   o  A Java 1.5 or later compatible virtual machine for your operating system.
